# Notes from Coding Jesus best videos

## Everything you need to know to become a quant trader (top 5 books)

Date: Jul. 2021

[Video Link](https://www.youtube.com/watch?v=FuNTNDEUwkw)

Summary:

- *Option Volatility and Pricing* by Sheldon Natenberg
- *Dynamic Hedging: Managing Vanilla and Exotic Options* by Nassim Taleb
- *Frequently Asked Questions in Quantitative Finance 2nd Ed.* by Paul Wilmott
- *Python for Data Analysis* by Wes McKinney
- *Introduction to Linear Algebra* by Gilbert Strang
- *Advances in Portfolio Management* by Richard Grinold and Ronald Kahn
- *Technical Analysis: Is Mostly Bullshit - Why Flipping a Coin is a Better Strategy than Using Technical Analysis in the Financial, Stock, and Forex Markets* by Tim Morris

## How to break into high frequency trading (as a software developer)

Date: Sep. 2020

[Video Link](https://www.youtube.com/watch?v=iSjNYhbrh1E)

Summary:

- Be proficient in C++
- Be able to solve interview questions, have projects in C++
- Learn a 2nd language such as C# or python
- Softskills: Communication and translating trader requirements to good features
- Softskills: Problem solving
- Misconception 1: You dont need to have trading background
- Misconception 2: You dont need to be a math genius
- "Passion, motivation, resiliance and ambition are much more important than being some unknown hacker elite billionaire trader"

This video is geared towards people looking to work in an HFT firm as software developers, as in, writing backend, frontend and data visualizations with code for traders that take decisions and make the money.
